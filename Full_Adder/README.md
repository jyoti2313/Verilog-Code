# Full Adder using Verilog HDL

## Overview

This project implements a 1-bit Full Adder using Verilog HDL.

A Full Adder adds three 1-bit binary inputs (A, B and Cin) and produces two outputs:
- Sum
- Carry Out (Cout)

---

## Inputs

- A
- B
- Cin

## Outputs

- Sum
- Cout
